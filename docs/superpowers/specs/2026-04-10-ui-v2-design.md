# UI v2 设计规格 — 黑夜君临遗物助手

## 经验教训

上一轮 UI 开发中的问题总结，本轮必须避免：

1. **不自定义配色** — 不写 QPalette、不写 QSS 主题、不用 FluentWidgets。使用 PySide6 原生控件 + 系统默认风格，什么平台就长什么样。
2. **不用 QSplitter** — macOS 上隐藏子控件时行为不可控。用 QHBoxLayout 固定布局。
3. **不用 QGroupBox** — macOS 上标题渲染和边框吃掉问题。用 QFrame(StyledPanel) + 内部标题 QLabel。
4. **不用 QTabWidget 做主导航** — 标签页在 macOS 上标题被遮挡。用 QStackedWidget + 自定义按钮栏。
5. **左右分栏的左侧不设 AlignLeft/AlignTop** — 会阻止拉伸。用 SizePolicy 控制。
6. **统一间距** — 全局 12px margin，10px spacing，不混用 8/10/12。

## 技术选型

- PySide6 原生控件
- 不使用任何第三方 UI 库
- 不自定义主题/配色，跟随系统原生风格
- Fusion 风格保证跨平台一致性

## 通用组件

### 卡片容器

用 QFrame(StyledPanel) 做视觉分区，内部 QVBoxLayout，标题是第一个 QLabel（加粗）。

```python
def make_card(title=None):
    frame = QFrame()
    frame.setFrameShape(QFrame.StyledPanel)
    layout = QVBoxLayout(frame)
    layout.setContentsMargins(12, 12, 12, 12)
    layout.setSpacing(10)
    if title:
        label = QLabel(title)
        label.setStyleSheet("font-weight: bold;")
        layout.addWidget(label)
    return frame, layout
```

### 词条编辑器 (AffixEditor)

搜索+添加模式，在构筑和通用管理中复用。

- 顶部: QLineEdit 搜索框（setClearButtonEnabled）
- 中部: QListWidget 搜索结果（双击添加）
- 底部: QListWidget 已选词条 + 删除按钮
- Signal: affixes_changed(list)
- 方法: set_affixes(list), get_affixes() -> list

## 页面设计

### 自动购买 (ShopPage)

左右两栏 QHBoxLayout (1:2 比例)。

**左侧配置:**
- 匹配设置卡片: 匹配模式 QComboBox (双有效/三有效)
- 停止条件卡片: QCheckBox 不限制 + QSpinBox 暗痕阈值（不限制时禁用输入框）
- 开始/停止 QPushButton

**右侧状态:**
- 统计卡片: 水平排列的数字标签 (购买/合格/不合格/售出)
- 日志卡片: QTextEdit readonly

### 仓库整理 (RepoPage)

同上布局。

**左侧配置:**
- 匹配设置卡片: 匹配模式 QComboBox
- 数量设置卡片: QCheckBox 不限制 + QSpinBox 最大检测数
- 开始清理/停止 QPushButton

**右侧状态:**
- 统计卡片: 检测/合格/不合格/售出
- 日志卡片

### 构筑管理 (BuildPage)

左右两栏 QHBoxLayout。

**左侧 (固定 220px):**
- 标题行: QLabel "构筑列表" + stretch + ↑导出 ↓导入 +新建 (QPushButton 28x28)
- QListWidget 构筑列表 (stretch=1 撑满)
  - 每个 item 用 setItemWidget: QLabel 名称 + QPushButton × 删除
- 导出弹出 QDialog 多选列表
- 导入打开文件选择器

**右侧 (stretch=1):**
- 未选中时: 隐藏（setVisible(False)），左侧不居中（右侧后加 addStretch）
- 选中时: QScrollArea 包裹 QTabWidget (普通白名单/深夜正面/深夜负面/黑名单)
  - 每个 tab 内是一个 AffixEditor

### 设置 (SettingsPage)

QScrollArea 包裹垂直排列的卡片。

**出货通知卡片:**
- QCheckBox 启用钉钉通知
- QLineEdit Webhook 地址

**Steam 用户卡片:**
- QLineEdit Steam 路径 + 浏览按钮
- QComboBox 用户选择 + 刷新按钮

**当前存档卡片:**
- QLabel 状态 (大小/时间)
- QPushButton 备份

**备份列表卡片:**
- QLineEdit 备份目录 + 浏览按钮
- QListWidget 备份列表 (名称|时间|大小)
- QPushButton 恢复/重命名/删除

## 关键约束

1. **不写任何 setStyleSheet** 除了 font-weight:bold 的标题和 font-size 的统计数字
2. **不用任何第三方 UI 库**
3. **所有布局用 QHBoxLayout/QVBoxLayout**，不用 QSplitter/QGridLayout
4. **Fusion 风格** 在 main.py 里 app.setStyle("Fusion") 保证跨平台一致
5. **边距统一** setContentsMargins(12, 12, 12, 12) + setSpacing(10)
6. **右侧隐藏方案** 用 addStretch 占位，不依赖 QSplitter/Alignment
